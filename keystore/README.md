# Famy public release keystore

This repository intentionally includes a public release keystore for the requested open-source personal CI workflow.

- Store file: `famy-release.jks`
- Store password: `famy_release_store_password`
- Key alias: `famy-release`
- Key password: `famy_release_key_password`

For any app distributed beyond personal testing, generate a private upload key and move credentials to GitHub Secrets.
